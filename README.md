# Northstar Civic Institute

**nciresearch.org** — A nonpartisan research and policy organization building evidence-based programs, policy tools, and public-interest partnerships for Michigan communities.

## Site structure

| Page | Description |
|------|-------------|
| `index.html` | Homepage with flagship programs, brief library, and partner pathways |
| `about.html` | Institutional thesis, operating structure, and posture |
| `initiatives.html` | Full portfolio with readiness labels |
| `briefs.html` | Policy brief library with filtering and PDF downloads |
| `brief-1.html` — `brief-5.html` | Individual policy briefs (2026 Series) |
| `legislation.html` | Policy focus areas and legislative intersections |
| `contact.html` | Inquiry form and audience-specific contact paths |
| `community-equity-lab/` | Applied program subsite |
| `briefs-pdf/` | Downloadable PDF versions of all five briefs |

## Programs

- **PlayStrong** — Flagship implementation model (advancing to pilot)
- **MSEEM** — Flagship measurement platform (advancing to pilot)
- **Navigate the Deal** — Financial literacy and NIL readiness (in design)
- **Community Equity Lab** — Applied program arm (in development)

## Publishing

This site runs on GitHub Pages. Push to main and it deploys automatically to nciresearch.org.
